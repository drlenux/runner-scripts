# runner-scripts

## Install

```bash
sh -c "$(wget https://raw.githubusercontent.com/drlenux/runner-scripts/master/install.sh -O -)"
```

## Run
```bash
runners
```

## Update
```bash
runners-update
```

## Unistall
```bash
runners-unistall
```

## require

- php `>= 5.6`
- composer
- ext-yaml
- ext-json
- ext-zlib
